# scripts

Executable code run from the book root directory — `.mjs`, `.py`, `.sh`, etc.

**Prompts do not live here.** Prompt sets and command suites live in
`prompts/`; `scripts/` is code only. (This resolves the old naming debt where
`scripts/` held `cowork-*.md` prompt files — those moved to `prompts/`.)

## build-prompts.mjs

Compiles a CLI-agnostic prompt suite in `prompts/<suite>/` into a tool-native
adapter. The suite's body + knowledge files are the tested rules and are never
altered by a build — only wrapped/packaged per target.

```bash
node scripts/build-prompts.mjs courses            # build every target in manifest
node scripts/build-prompts.mjs courses --target skill
```

Output lands in `prompts/<suite>/.build/` (git-ignored). Targets implemented:
`skill` (Claude Code / Cowork `.skill`). `agents` (Codex `AGENTS.md`) and
`cursor` (`.cursor/rules`) are stubbed — add a builder function. If a target
requires a manifest field that is missing, the build stops and names the field
rather than guessing a default.

## svg-to-png.mjs

Converts every `images/**/*.svg` to a 300dpi PNG. Idempotent — skips PNGs that
are already newer than their SVG source.

```bash
node scripts/svg-to-png.mjs
```

**Requires:** `sharp` and `glob`

```bash
npm install sharp glob
```

`sharp` depends on `librsvg` for SVG rendering. All SVGs generated by Cowork
embed their fonts as base64 — no system font dependency.
